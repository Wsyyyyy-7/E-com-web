# Project Summary
CreatorBridge is a collaborative platform that connects creators with businesses through a marketplace model. It allows creators to showcase their work, manage campaigns, and handle contracts, while enabling businesses to discover and collaborate with a diverse range of creators. The platform features user profiles, a marketplace for campaigns, and tools for managing interactions and contracts, enhancing the overall experience for both parties. Recent updates have focused on enhancing the visual design and user engagement across the platform, particularly with a unified dark theme.

# Project Module Description
- **Backend**: Manages business logic, authentication, database interactions, and API endpoints.
- **Frontend**: Provides an intuitive interface for users with responsive design, now featuring a dark theme across all pages.
- **Data Models**: Structures for user profiles, campaigns, messages, and contracts.
- **Services**: Handles application logic related to campaigns and user interactions.
- **Routers**: Manages API routes for features such as authentication and data retrieval.

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build the Project**:
   - For the backend: Run `python main.py`.
   - For the frontend: Run `pnpm run build`.

3. **Run the Project**:
   - For the backend: Start the server using the command for your framework.
   - For the frontend: Run `pnpm run dev`.
